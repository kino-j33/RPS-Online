# Janken Web Application

## 概要
職業訓練の個人課題として作成した、Java（Servlet/JSP）を用いたじゃんけんWebアプリケーションです。  
HTTPリクエスト・レスポンスの流れや、サーバーサイドでの処理を理解することを目的としています。

---

## アプリ内容
ブラウザ上で手（グー・チョキ・パー）を選択し、  
サーバー側で生成された手と比較して勝敗を表示します。

### 主な機能
- じゃんけんの手を選択して送信
- サーバー側でランダムに手を生成
- 勝敗結果を画面に表示

---

## 使用技術
- Java  
- Servlet / JSP  
- HTML / CSS  
- Apache Tomcat  
- Eclipse  

---

## 画面・動作イメージ
1. トップ画面で手を選択  
2. 「勝ち / 負け / あいこ」を結果画面に表示  

---

## 工夫した点
- 勝敗判定のロジックをメソッドとして切り出し、可読性を意識しました
- JSPではJavaコードを最小限にし、Servlet側で処理を行う構成にしました
- 定数を使用してマジックナンバーを減らしました
